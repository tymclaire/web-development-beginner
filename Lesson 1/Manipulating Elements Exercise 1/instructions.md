# Manipulating Elements Exercise 1

## Instructions

For the JavaScript to manipulate the DOM, we shall start with JavaScript code inside the HTML document
The JavaScript Code will be inside `<script> </script>` tags

Edit the code within the script tags

Try:

- Select all the elements with class = "print" and change the inner text content to “Hi there, this is a print element!”

- Use setAttribute to change the href value of the <a> element to 'https://www.google.com’

- Using style property to change inline styles of the element with id = “header” so that the font is blue and background color is light gray

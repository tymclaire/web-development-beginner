# Fetch API Exercise 1

## Instructions

In this exercise, you will complete the JavaScript code to dynamically display a list of countries on a web page. The HTML file is already provided, and your task is to finish the provided JavaScript code.

- The HTML file has been set up as index.html
- The JavaScript file has been set up as script.js
- You will need to complete the forEach loop to create the list items and add each country's name it it

### Steps

- Step 1: Create a list item ```<li>``` element using document.createElement('li').
- Step 2: Set the textContent of the list item to the country's name using country.name.
- Step 3: Append the list item to the unordered list using ul.appendChild(li).

### Expected Behavior

When the code runs, the HTML page should have a list of all the country names. The console should also display the data from the API.

## Running the Test

To test if your code is correct, open the HTML file in Chrome and refresh it.

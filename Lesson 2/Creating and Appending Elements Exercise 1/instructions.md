# Creating and Appending Elements Exercise 1

## Instructions

For the JavaScript to manipulate the DOM, we shall start with JavaScript code inside the HTML document
The JavaScript Code will be inside `<script> </script>` tags

Edit the code within the script tags

Try:

- Create a paragraph element that says, “This is a fifth paragraph.”

- Create an unordered list with 2 list elements and append it to the bottom of the document

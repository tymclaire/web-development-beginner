# Asynchronous JavaScript Exercise 1

## Instructions

In this exercise, students will understand how asynchronous JavaScript executes code, focusing on async functions like setTimeout and the event loop.

- Look at the code.
- In what order will the the text be printed in the console?
- Why does "Hello, Ninja!" appear last even though setTimeout is called earlier?
- Try increasing or decreasing the delay in setTimeout.

## Running the Test

To test if your code is correct, run the test with the following commands in terminal: ```node test.js```

# Removing Event Listeners Exercise 1

## Instructions

For the JavaScript to manipulate the DOM, we shall start with JavaScript code inside the HTML document
The JavaScript Code will be inside `<script> </script>` tags

Edit the code within the script tags

Try:

- Task 1: Remove the click event listener
  - Write JavaScript to remove the existing click event.
  
- Task 2: Implement a new click event listener
  - Write JavaScript to add a new paragraph element before the button, after every click.

Note: The base functions ```alertText``` (for task 1) and ```addText``` (for task 2), have been written for you in the code.

## Running the Test

To test if your code is correct, open the HTML file in Chrome.
Now refresh the page!

Make sure the correct elements respond when clicked, and moused over.

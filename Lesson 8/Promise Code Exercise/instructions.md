# Promise Code Exercise

## Instructions

Complete the section of code in main.js for resolve and reject.

Task 1

- Add in the appropriate code to change the resolve value to "The coin landed on heads!"
- Add in the appropriate code to change the reject value to "The coin landed on tails!"

When the code runs, the console should display "The coin landed on heads!" or "The coin landed on tails!" at random.
